To quickly explain it, first, you load jQuery and the jquery.graphup.pack.js file in the head of your HTML document. Then you can call the graphup() method on the table cells you select.

$('#table td').graphup({
	// set options here
});


Note: the three demos included in the download are PHP files. PHP is used just to generate some random data tables quicker. If you can't run PHP for some reason, you can still check out the hosted demos:
- http://www.geertdedeckere.be/shop/graphup/download/demo1/
- http://www.geertdedeckere.be/shop/graphup/download/demo2/
- http://www.geertdedeckere.be/shop/graphup/download/demo3/
